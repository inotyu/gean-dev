# Gean Carlos - Portfólio Pessoal

Portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro. Um site moderno e responsivo apresentando minhas habilidades, projetos e experiência profissional.

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização moderna com variáveis CSS
- **JavaScript** - Interatividade e animações
- **Google Fonts** - Tipografia Sora

## 📁 Estrutura do Projeto

```
gean-dev/
├── index.html          # Página principal
├── style.css          # Estilos CSS
├── script.js          # JavaScript
└── src/               # Recursos adicionais
    ├── Jogos/         # Jogos desenvolvidos
    ├── animes/        # Projetos relacionados
    └── profile/       # Imagens de perfil
```

## 🏃 Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/inotyu/gean-dev.git
```

2. Abra o arquivo `index.html` no navegador ou use um servidor local:
```bash
# Usando Python
python -m http.server 8000

# Usando Node.js
npx serve .

# Ou simplesmente abra index.html no navegador
```
